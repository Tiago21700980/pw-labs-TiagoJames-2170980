Elemento foi usado? Explicação
Universal star aplica o estilo a todos os elementos da pagina
Tipo star aplica o estilo do nome do marcador a utilizar
Classe visibility_off aplica o estilo do nome de uma classe a utilizar
Identificador star aplica o estilo consoante o Identificador(muito util em Javascript)
Atributos star aplica o estilo consoante o atributo de um Elemento
Pseudo-classe visibility_off aplica o estilo consoante o estado de um elemento
Pseudo-elemento visibility_off aplica o estilo consoante em partes de elementos
